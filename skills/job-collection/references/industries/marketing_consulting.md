# 广告营销与咨询

## 行业定位

覆盖广告代理、公关、营销咨询、品牌策划、4A 公司、本土营销集团、咨询公司等以服务客户为核心的机构。该行业校招以"管培 / 培训生"为主流入口，常见项目名（奥美 Fellow、群邑 MT、Interpublic 培训生、埃森哲 Analyst 等）伴随明确的客户组分配，需把"客户行业"作为强字段维护。

## 默认信息源

- 公司官网招聘页
- 4A 与本土营销集团公众号
- 高校就业网营销咨询专场
- 行业垂直招聘公众号
- 用户自定义信息源

## 行业扩展字段

- 机构类型（4A / 本土营销集团 / 公关 / 营销咨询 / 品牌策划 / 战略咨询 / 数字营销）
- 业务条线（客户管理 / 创意 / 策略 / 数据 / 媒介 / 内容）
- 客户行业（FMCG / 汽车 / 互联网 / 金融 / 医疗 等）
- 是否管培项目
- 项目名称
- 是否涉及海外客户
- 工作地点（含总部 / 区域）

## 面试准备重点

- 公司业务结构与客户矩阵
- 业务条线商业模式
- 客户行业洞察与 case 拆解
- 创意 / 策略 / 数据能力
- 实习经历与岗位匹配点

## Offer 评价重点

- 客户矩阵与项目类型
- 业务条线前景
- 薪资结构与加班文化
- 工作强度
- 城市与生活成本
- 晋升路径

# 医疗与健康

## 行业定位

覆盖医药、医疗器械、生物科技、CRO/CDMO、医院、诊所、医疗保险等机构。该行业校招按"研发驱动 / 销售驱动 / 临床驱动"分层明显，常见项目名（辉瑞 GIP、罗氏 PD、迈瑞校招、恒瑞管培等）与岗位序列强绑定，需区分应届项目与常规校招。

## 默认信息源

- 公司官网招聘页
- 跨国药企 / 本土药企公众号
- 高校就业网医疗专场
- 行业垂直招聘公众号（医药代表网、医脉同道等）
- 用户自定义信息源

## 行业扩展字段

- 机构类型（跨国药企 / 本土药企 / 器械 / CRO/CDMO / 生物科技 / 医院 / 医保）
- 业务条线（研发、临床、注册、市场、销售、医学事务、医保准入）
- 是否管培项目
- 项目名称
- 是否需要临床/医学背景
- 是否涉及海外业务
- 工作地点（含总部 / 区域 / 医院）

## 面试准备重点

- 公司管线与主要产品
- 业务条线商业模式
- 行业政策（集采、医保、注册审评）
- 临床/医学知识储备（如适用）
- 实习经历与岗位匹配点

## Offer 评价重点

- 公司管线前景与产品组合
- 业务条线前景
- 薪资结构与奖金机制
- 工作强度
- 城市与发展路径
- 平台稳定性

# 教育

## 行业定位

覆盖 K12、职业教育、教育科技、出版、高校行政、教培服务等机构。该行业校招按"教研驱动 / 销售驱动 / 产品驱动"分层，"双减"后业务重心明显转向职业教育、素质教育和教育科技，需关注业务合规边界。

- 业务类型（K12 / 职教 / 素质 / 教育科技 / 出版 / 高校行政 / 教培服务）

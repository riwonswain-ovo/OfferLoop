# 文化传媒与娱乐

## 行业定位

覆盖影视、游戏、直播、短视频、数字文创、出版、动漫、音乐等以内容生产与分发为核心的机构。该行业校招按"内容创意 / 平台运营 / 商业化"分层，常见项目名（腾讯视频"创造营"、字节星辰、bilibili 校招等）伴随明确的业务线分配，需把"内容垂类"作为强字段维护。

## 默认信息源

- 公司官网招聘页
- 内容平台与影视公司公众号
- 高校就业网文化专场
- 行业垂直招聘公众号
- 用户自定义信息源

## 行业扩展字段

- 业务类型（影视 / 游戏 / 直播 / 短视频 / 出版 / 动漫 / 音乐 / 数字文创）
- 内容垂类（如游戏品类、影视类型、二次元、知识、生活）
- 业务条线（内容、运营、商业化、产品、研发）
- 是否管培项目
- 项目名称
- 工作地点（含总部 / 区域）
- 是否涉及出海内容

## 面试准备重点

- 公司业务结构与内容矩阵
- 业务条线商业模式
- 内容垂类的受众与市场
- 行业政策（版号、内容监管、未成年人保护）
- 实习经历与岗位匹配点

## Offer 评价重点

- 业务条线前景
- 内容垂类的成长空间
- 薪资结构
- 工作强度
- 城市与生活成本
- 平台稳定性

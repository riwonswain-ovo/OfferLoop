# 新能源与汽车

## 行业定位

覆盖新能源整车、动力电池、光伏、储能、氢能、芯片半导体、智能驾驶、汽车零部件等机构。该行业近两年扩张极快，校招有大批量"研究/工程/技术"序列岗位，常见项目名（蔚来 UT、小鹏 NGP、宁德时代钛钠、隆基绿能等）伴随明确的城市分配与补贴政策，需把"补贴城市 / 落户支持"作为强字段维护。

## 默认信息源

- 公司官网招聘页
- 企业公众号
- 高校就业网新能源专场
- 牛客汽车/能源板块
- 行业垂直招聘公众号
- 用户自定义信息源

## 行业扩展字段

- 细分赛道（整车 / 电池 / 光伏 / 储能 / 氢能 / 半导体 / 智能驾驶 / 零部件）
- 业务条线（研发、工程、制造、供应链、销售、智能驾驶、算法）
- 是否管培项目
- 项目名称
- 工作基地（含研发中心 / 工厂 / 总部）
- 是否涉及海外 / 出海业务
- 是否有落户 / 购房补贴
- 是否涉及芯片国产化方向

## 面试准备重点

- 公司技术栈与产业链位置
- 智能驾驶/算法方向的技术趋势
- 行业政策导向（补贴、碳中和、国产替代）
- 实习经历与岗位匹配点

## Offer 评价重点

- 细分赛道前景
- 工作基地与生活成本
- 薪资与补贴结构（含购房/落户）
- 工作强度
- 期权 / 股票
- 平台稳定性与扩张速度

# 互联网与科技

## 行业定位

覆盖互联网平台、科技企业、电商与物流、通信与智能终端设备等以数字化业务为主的机构。该行业内校招节奏快、批次密集，常见细分项目名（如京东 JDS/JDY/TET、阿里星图/青藤、腾讯技术大咖），需按项目名而非岗位名维护招聘批次。

## 默认信息源

- 企业校招官网
- 互联网公司招聘官网
- 牛客、Boss直聘等校招信息平台
- 高校就业网
- 官方公众号
- 用户自定义招聘链接

## 行业扩展字段

- 业务线
- Base 城市
- 是否提前批
- 是否内推
- 内推码 / 内推链接
- 是否可转正
- 是否有笔试
- 招聘项目名（JDS / TET / 星图 / 青藤等）

## 面试准备重点

- 公司核心业务与商业模式
- 业务线分工与各自前景
- 产品体验与竞品分析
- 用户增长、商业化、数据分析能力
- 简历项目与岗位要求的匹配点

## Offer 评价重点

- 薪资水平与期权
- 业务线前景
- 成长空间
- 工作强度
- 城市与生活成本
- 平台价值

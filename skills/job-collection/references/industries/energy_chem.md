# 能源与化工

## 行业定位

覆盖石油、化工、电力、煤炭、钢铁、有色金属等以能源与重化工为主的机构。该行业校招以国企/大型集团为核心，常见项目名（中石化青苗、中海油海之子、国家电网校招等）与编制/落户/政审体系强绑定，需把"集团层级与基地分配"作为强字段维护。

## 默认信息源

- 公司官网招聘页
- 集团与子公司公众号
- 高校就业网能源专场
- 行业垂直招聘公众号
- 用户自定义信息源

## 行业扩展字段

- 细分赛道（石油 / 化工 / 电力 / 煤炭 / 钢铁 / 有色金属 / 核电）
- 上级集团与单位层级
- 业务条线（勘探、生产、研发、销售、贸易、管理）
- 是否管培项目
- 项目名称
- 工作基地（含总部 / 油田 / 厂区 / 区域公司）
- 是否涉及政审 / 体检
- 是否涉及海外（含海外油气项目）

## 面试准备重点

- 集团结构与产业链位置
- 业务条线商业模式
- 行业政策（双碳、能源安全、国企改革）
- 政审 / 体检要求
- 实习经历与岗位匹配点

## Offer 评价重点

- 单位层级与编制性质
- 工作基地与生活成本
- 薪资与补贴结构
- 户口 / 落户支持
- 工作强度与基地条件
- 平台稳定性
